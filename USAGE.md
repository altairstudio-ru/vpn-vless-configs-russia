# Usage Guide

## How to Use These Proxy Configurations

This repository contains regularly updated free and paid proxy configurations for VLESS, VMESS, and Hiddify protocols.

### Supported Protocols

- **VLESS** - Fast and lightweight proxy protocol
- **VMESS** - Feature-rich proxy protocol  
- **Hiddify** - Optimized for streaming and reliability

### Getting Started

1. **Download the configurations**: Clone this repository or download the files directly
2. **Import configurations**: Use any compatible proxy client (Clash, V2Ray, Shadowrocket, etc.)
3. **Select a proxy**: Choose from the available servers based on your location and speed needs
4. **Connect**: Start using the internet anonymously

### Configuration Files

- `subscriptions/all_base64.txt` - All configurations in base64 format
- `data/` - Individual configuration files
- `vpn-files/` - Additional proxy lists
